This page is about miscellaneous HTML things, but you may also be interested in links or images.

Character Sets

You can use UTF-8 to count in Turkish:

  1. bir
  2. iki
  3. üç
  4. dört
  5. beş

HTML Entities

There are three reserved characters in HTML: < > and &. You should always use HTML entities for these three characters.

If you’re into “web typography,” you’ll also find yourself using curly quotes a bit.
Since we are now usinf a UTF-8 character set, we should be able to insert any character directly into the HTML document.
