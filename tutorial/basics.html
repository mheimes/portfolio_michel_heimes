Headings

Headings define the outline of your site. There are six level of headings.

Lists:

This is how you make an unordered list:

  • Add a "ul" element (it stands for unordered list)
  • Add each item in its own "li" element
  • They don't need to be in any particular order
  • Never add any other elements directly in an unordered list apart from "li" elements!
    • Test - unordered list within an unordered list
      • Again an unordered list within an unordered list, within an unordered list.

  • Instead, you should wrap paragraphs etc. with "li" tags.
